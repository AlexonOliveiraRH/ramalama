## NAME
ramalama - Run specified AI Model as a chatbot

## SYNOPSIS
**ramalama run** [*options*] *model*

## DESCRIPTION
Run specified AI Model as a chat bot. Ramalama pulls specified AI Model from
registry if it does not exist in local storage.
